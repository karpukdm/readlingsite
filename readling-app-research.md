# Readling — Исследование приложения

## Общая концепция

**Readling** — мобильное приложение для изучения английского языка через чтение настоящих книг. Вместо зубрёжки и упражнений пользователь читает интересную литературу, а язык усваивается естественно — через контекст.

**Платформы**: iOS, Android (React Native / Expo)
**Версия**: 1.6.1
**Бэкенд**: Spring Boot + PostgreSQL + Supabase

---

## Ключевые фичи

### 1. Два режима чтения

#### Режим погружения (Immersion / Hybrid)
- Английские фразы постепенно вплетаются в русский текст
- На первых страницах ~90% русского, ~10% английского
- С каждой страницей доля английского растёт
- Мозг привыкает к английскому постепенно, без стресса
- **Это уникальная фича — аналогов на рынке практически нет**

#### Параллельное чтение (Parallel)
- Английский оригинал и русский перевод рядом
- Удобно для сравнения конструкций и грамматики
- Классический подход, знакомый пользователям

### 2. Интерактивное чтение
- **Тап по слову/фразе** — мгновенный перевод
- **Автосохранение** — все посмотренные слова автоматически попадают в личный словарь
- **Аудиопроизношение** — можно послушать как звучит слово
- **Настройки текста** — шрифт, размер, межстрочный интервал, тема (светлая/тёмная/сепия)
- **Навигация по главам** — быстрый переход через оглавление

### 3. Три типа тренировок

#### Флэш-карточки
- Повторение сохранённых фраз
- Система свайпов (знаю / не знаю)
- Адаптивное повторение

#### Собери предложение (Sentence Puzzle)
- Перемешанные слова нужно расставить в правильном порядке
- 3 уровня сложности: EASY, MEDIUM, HARD
- Тренирует грамматику и порядок слов

#### Вставь слово (Fill-in-the-Word)
- Предложение с пропущенным словом
- Нужно собрать слово из перемешанных букв
- Тренирует правописание и запоминание

### 4. Каталог книг
- Книги с уровнями сложности A1–C2 (CEFR)
- 25 жанров (фантастика, детективы, романтика, классика, поэзия, детские и др.)
- Поиск и фильтрация
- Рейтинги и отзывы пользователей
- Возможность запросить перевод новой книги

### 5. Геймификация и мотивация
- **Стрики** — ежедневная серия чтения
- **Трекинг активности** — прочитанные главы, время чтения, тренировки
- **Прогресс по книгам** — визуальный прогресс-бар

### 6. Личный словарь (Phrase Cards)
- Все фразы в одном месте
- Вкладки: изучаемые / выученные / все
- Интеграция с тренировками — учишь именно свои слова

### 7. Библиотеки и коллекции
- Пользовательские подборки книг
- Системные рекомендации на основе истории чтения

### 8. Социальные функции
- Реферальная программа (обычные и коммерческие коды для блогеров)
- Система обратной связи с ответами от команды
- Запрос на добавление новых книг

## Конкурентные преимущества (УТП)

### 1. Уникальный режим плавного погружения
Ни одно популярное приложение не предлагает постепенное замещение родного языка иностранным прямо внутри текста книги. Это запатентованный по сути подход: читаешь книгу на русском, а английский "прорастает" органично.

### 2. Учёба через удовольствие
Не упражнения и не зубрёжка — настоящие книги. Пользователь совмещает хобби (чтение) с изучением языка. Мотивация не выгорает, потому что хочется узнать, чем закончится история.

### 3. Контекстное запоминание
Слова запоминаются в контексте реальных предложений из книг, а не изолированно. Это научно более эффективный подход к запоминанию лексики.

### 4. Персонализированные тренировки
Тренировки строятся на словах, которые пользователь сам посмотрел при чтении — то есть на его реальных пробелах в знаниях, а не на случайных списках.

### 5. Градация сложности A1–C2
Полный спектр от начинающих до продвинутых. Можно расти внутри приложения, переходя от простых книг к сложным.

### 6. Множественные способы закрепления
Одно слово закрепляется через: чтение в контексте → флэш-карточки → сборка предложения → вставка слова. Разные каналы памяти задействованы.

### 7. Нативная поддержка iOS и Android
Единая кодовая база (React Native/Expo), но нативный UX на обеих платформах.

## Дизайн-концепция

### Цветовая палитра
- **Primary**: синий (#007AFF iOS / #0A84FF dark)
- **Фон**: нейтральный серый (светлая), тёмный (#141414)
- **Поверхности**: белый / #1E1E1E (dark)
- **Тема чтения**: сепия (для комфортного чтения)

### Типографика
- System fonts (SF Pro Display на iOS)
- Display: 24pt bold
- Heading: 20pt semibold
- Body: 16pt regular, line-height 24pt
- Caption: 14pt

### Принципы UI
- Material Design 3
- 8pt grid система
- Минимальный touch-target: 44×44pt
- Скругления: 4–28px (зависит от элемента)
- Полная поддержка VoiceOver/TalkBack (accessibility)

### Анимации
- "Worm effect" на page indicator
- Плавный прогресс-бар (420ms)
- Scale-down при нажатии кнопок (0.985, 120ms)
- Staggered появление карточек (40ms задержка)

## Монетизация

### Подписка (RevenueCat)
- **Месячная** подписка
- **Годовая** подписка (рекомендуемая, с пометкой "выгоднее")

### Что даёт подписка
- Доступ ко всем книгам (включая платные)
- Без рекламы
- Все режимы перевода
- Приоритетная поддержка (годовой план)

### Промокоды
- Коммерческие (для блогеров/инфлюенсеров)
- Некоммерческие (для акций)
- Трекинг активаций

## Целевая аудитория

1. **Начинающие (A1–A2)** — хотят начать учить английский без стресса
2. **Средний уровень (B1–B2)** — хотят расширить словарный запас через чтение
3. **Продвинутые (C1–C2)** — хотят читать оригиналы с поддержкой
4. **Любители чтения** — совмещают хобби с пользой
5. **Те, кому не подошёл Duolingo** — ищут менее "игровой" и более "содержательный" подход

## Контент-модель

### Поддерживаемые форматы книг
- EPUB (через EPUBlib)
- FB2 (кастомный парсер — популярный формат в РФ)
- HTML

### Структура данных
- **Книга** → главы → параграфы → предложения
- Каждый уровень хранит оригинал + перевод
- Предложения используются для генерации тренировок-пазлов
- Обложки хранятся в Yandex S3

### Перевод
- Интеграция с OpenAI API для автоматизации перевода
- Хранение переводов на уровне фраз (для hybrid mode) и предложений (для parallel)
